Visual studio code is an offline application and does not need internet to work.You can download it from the internet or app store and start working as soon as it is donwloaded

The console screen is a screen used for logging things visually.When running a code, you can't see the numericals in the code. but if after every step you log the number in the console, the number will be visible. You can open the console by first right-clicking, then selecting inspect and finally clicking the console tab
